Sistema de Análise de Vendas em Python

Projeto desenvolvido para análise de dados de vendas utilizando Python.

Funcionalidades:

- Cálculo de faturamento total
- Identificação do produto mais vendido
- Cálculo de ticket médio
- Geração de relatório simples

Tecnologias:

- Python

Objetivo:

Praticar lógica de programação e análise de dados.
